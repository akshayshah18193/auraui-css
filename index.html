AuraUI

A lightweight, plug-and-play CSS framework for building modern interfaces with glass morphism, token-driven theming, a complete form kit, and a responsive layout system. Zero JavaScript dependencies — drop one file into any project.

Everything you need

A full component library in a single CSS file, designed to be dropped into any project.

🪟

Glass Morphism

Three glass tiers (au-glass-sm, au-glass, au-glass-lg) with configurable blur and opacity via CSS custom properties.

📝

Complete Form Kit

Inputs, selects, checkboxes, radios, toggles, sliders, file uploads, and all validation states — fully accessible.

🧩

Responsive Layout

Responsive grid, flex utilities, auto-fit columns, and a 12-column system — all in pure CSS, no framework required.

🎨

Token-Driven

Every color, radius, and spacing value is a CSS custom property. Theme the whole system from one :root block.

🌗

Dark + Light Themes

Toggle between light and dark themes with a single au-dark class on the body element. No JavaScript needed.

🎛️

Live Playground

Visual customizer to design your theme and export it as a ready-to-use CSS override file. Instant live preview.

📦

~5 KB Gzipped

No dependencies, no build step, no runtime. One CSS file, works anywhere — plain HTML, React, Vue, Svelte, Angular, or any bundler.

♿

Accessible by Default

Focus rings on every interactive element, reduced-motion support, and semantic markup patterns baked in.

Quick start

Add AuraUI to any project in seconds.

1 · HTML (CDN or local file)
<link rel="stylesheet" href="aura-ui.css">
2 · Use components
<button class="au-btn au-btn-primary">Get started</button>

<div class="au-glass au-p-6">
  <h3>Glass card</h3>
  <p>Frosted-glass surface.</p>
</div>

<input class="au-input" type="text" placeholder="Type here…">
3 · Customise tokens (optional)
/* Override any token in your own stylesheet */
:root {
  --au-color-primary: #0ea5e9;   /* sky blue */
  --au-radius-lg: 20px;          /* rounder cards */
  --au-glass-blur: 24px;         /* heavier glass */
}

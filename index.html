Glass Morphism

Three glass tiers with multi-layer gradients, realistic borders, and highlight overlays for authentic frosted-glass effects. Fully customizable via CSS tokens.

📝

Complete Form Kit

Inputs, selects, checkboxes, radios, toggles, sliders, and file uploads with validation states — all fully accessible and keyboard-navigable.

Responsive Layout

Flexible grid, auto-fit columns, and a 12-column system with mobile-first breakpoints. Build any layout, pure CSS.

Quick start

Add AuraUI to any project in seconds.

1 · HTML (CDN or local file)
<link rel="stylesheet" href="glass.css">
2 · Use components
<button class="au-btn au-btn-primary">Get started</button>

<div class="au-glass au-p-6">
  <h3>Glass card</h3>
  <p>Frosted-glass surface.</p>
</div>

<input class="au-input" type="text" placeholder="Type here…">
3 · Customise tokens (optional)
/* Override any token in your own stylesheet */
:root {
  --au-color-primary: #0ea5e9;   /* sky blue */
  --au-radius-lg: 20px;          /* rounder cards */
  --au-glass-blur: 24px;         /* heavier glass */
}
